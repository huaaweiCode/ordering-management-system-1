# 订餐管理系统1

#### 介绍
订餐管理系统拥有两个角色：管理员和用户

管理员：添加菜品、修改菜品、修改个人信息、设置首页推荐菜品

用户：登录注册、购买菜品、查看已消费和未消费的菜、查看购物车、修改个人信息和送货地址
